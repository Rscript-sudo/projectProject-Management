# 项目文档管理系统 v1.0.0 · 首发版本

> 2026-06-22 · 首版发布

---

## 🎯 这是什么

面向监理行业的**项目文档一体化管理系统**。所有能力装在一个 Electron 应用里，本地运行，数据不出本机。

---

## ✨ 核心功能

### 📄 AI 文档生成（21 类模板）

覆盖监理业务全场景：

| 类别 | 模板 |
|------|------|
| **日常文档** | 监理日志 / 周报 / 月报 / 会议纪要 |
| **现场管控** | 监理整改通知书 / 联系单 / 节假日安全通知 / 开工条件检查表 |
| **报审签认** | 承建资格报审表 / 施工组织设计报审表 / 总监理工程师任命书 |
| **进度管控** | 工程变更单 / 开工通知 / 竣工通知 / 停工令 |
| **投资管控** | 工程款支付证书 / 进度分析报告 |
| **专项工程** | 通信工程 / 信息化工程 / 电力工程 / 监理规划 / 通用模板 |

- **流式 AI 生成**（SSE 协议，120 秒超时）
- **智能字段映射**（自然语言 → 表单字段，自动匹配）
- **Docx 标准排版**（GB/T 9704-2012 党政公文标准）

### 🔍 全文检索

- Docx / Xlsx 文档全文索引
- 跨项目、跨文件夹秒级搜索

### 📁 往来函件台账

- 整改 / 安全 / 联系单 / 函件 / 停工令 / 支付证书
- 自动归档到 `原始稿/` 和 `签认件/` 两个目录
- 5 维度检索：日期 / 类型 / 收发单位 / 编号 / 关键词

### ⚠️ 隐患联动

- 现场巡视 → 隐患自动入台账
- 隐患 → 一键生成整改通知
- 整改回执 → 关闭隐患，全流程闭环

### 📊 进度控制

- 横道图（手写 SVG，无需额外库）
- 偏差分析（计划 vs 实际）

### 💰 投资控制

- 5 级审批流
- 累计金额自动汇总
- 与支付证书模板联动

### 📜 合同管理

- 合同 / 变更 / 索赔台账
- 到期前 30 天自动预警

### 📷 照片归档

- 拖拽上传
- 按月自动归档
- EXIF 信息提取

---

## 🔧 技术栈

| 层 | 技术 |
|----|------|
| 桌面框架 | Electron 42 |
| UI | React 19 + Ant Design 5 + TypeScript |
| 构建 | Vite 6 |
| 数据库 | SQLite（better-sqlite3，WAL 模式） |
| 文档 | docx / docxtemplater / mammoth |
| AI | 流式 SSE 接入 |

## 📦 安装

### Windows 安装包（推荐）

下载 **`项目文档管理系统 Setup 1.0.0.exe`**，双击安装，按向导操作。

### Windows 绿色版

下载 **`项目文档管理系统 1.0.0.exe`**，双击即用，无需安装。

### 系统要求

| 项 | 要求 |
|----|------|
| OS | Windows 10 / Windows 11（**不支持 Win 7/8**） |
| 架构 | x64 |
| 磁盘 | 至少 500 MB 可用空间 |
| 内存 | 建议 4 GB 以上 |
| 网络 | 仅 AI 生成功能需要联网 |

## ⚠️ 已知限制

1. **未做代码签名** —— 首次安装时 Windows SmartScreen 会拦截，点"更多信息 → 仍要运行"即可
2. **AI 模型配置** —— 需在「设置」页面填入 API Key 才能用 AI 生成功能
3. **单项目单窗口** —— 当前不支持多开实例
4. **杀毒软件** —— 360 / 火绒可能误报，加入白名单即可
